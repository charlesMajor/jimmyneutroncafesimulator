# jimmyneutroncafesimulator

Projet final d'un cours de jeu vidéo sur unity: Café simulator

Build jouable : Build/Jimmy Neutron's Cafe Simulator.exe

Commandes : w, a, s,d pour les déplacements. Espace pour une roulade, E sur les clients pour les commandes et E sur les machines pour produire ou ramasser les produits
Tables déplaçables en maintenant E
